## Design Tools & Resources

### Recommended Tools:
1. **Figma** (Professional, free tier available)
   - Best for: All designs
   - Template: Import from Figma Community "Mobile App Ad Templates"

2. **Canva Pro**
   - Best for: Quick creation, has phone mockup templates
   - Search: "App promotion Instagram post"

3. **Adobe Express**
   - Best for: Quick social media ads
   - Has built-in brand kit feature

### Assets Needed:
1. **Phone Mockups**:
   - mockuphone.com (free)
   - Figma plugin: "Mockup" by Artify
   - Canva built-in mockups

2. **App Screenshots**:
   - Take screenshots from running app
   - Use Flutter's screenshot capability
   - Ensure screenshots show:
     - Dashboard with cards
     - Invoice list
     - Analytics charts
     - Payment interface

3. **Icons**:
   - Use Material Icons (already in your app)
   - Iconscout for additional icons
   - Flaticon for decorative elements

4. **Fonts** (already in your app):
   - Poppins (Google Fonts)
   - Inter (Google Fonts)
   - JetBrains Mono (for numbers)

## Quick Start Instructions

### Option 1: Canva (Easiest)
1. Go to Canva.com
2. Search "App Promotion Instagram Post"
3. Choose a template with phone mockup
4. Replace colors with InvoiceFlow palette
5. Add your app screenshots to phone
6. Update text with provided copy
7. Download as PNG (high quality)

### Option 2: Figma (Professional)
1. Create new file, set canvas size
2. Add gradient background using color stops
3. Use "Mockup" plugin to add phone
4. Import app screenshots
5. Add text layers with Google Fonts
6. Add effects (shadows, blurs)
7. Export as PNG @2x

### Option 3: Adobe Express
1. Choose "Social Media Post" template
2. Upload app logo and screenshots
3. Use gradient background tool
4. Add text overlays
5. Apply your brand colors
6. Download high-res PNG
